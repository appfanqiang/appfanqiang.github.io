第一章：资源经济学——战争机器的燃料库

1.1 采集器的战略定位

金矿与圣水采集器绝非简单的装饰物，它们的摆放堪称"生产力几何学"。顶级玩家常采用"三环嵌套法"：将采集器呈同心圆分布，外层放置低级采集器作为诱饵，核心圈保护高等级设施。这种设计既能延缓敌人掠夺效率，又能通过牺牲外围建筑为防御塔争取输出时间。

1.2 黑油的特殊管理学

暗黑石油的稀缺性决定了它必须享受"总统级安保"。建议将暗黑油井与防空火箭、X连弩组成"黄金三角"，利用防空力量克制常见的矿工流/气球流抢油战术。记住：被夺走的每滴黑油，都可能成为对手龙骑士碾压你村庄的催化剂。

第二章：兵种交响乐——毁灭艺术的配器法则

2.1 版本答案的辩证认知

2.3 英雄养成的优先级序列

野蛮人之王应优先升至50级解锁"铁拳出击"的质变技能，而咏王的关键节点在20级获得"永恒书卷"。有个被低估的技巧：在升级英雄期间，将其部署在建筑大师基地仍可参与对抗赛，这是保持战力的隐秘诀窍。

3.1 进攻前的"X光扫描"

FAQ

Netch 全局代理与规则代理的区别是什么？
全局代理会将所有流量通过节点转发，适合游戏或全局科学上网；规则代理只针对指定应用或 IP 走节点，优化带宽利用和网络效率，根据不同需求选择使用模式。
Sing-box 的策略组如何实现按域名分流？
Sing-box 支持根据域名规则匹配流量，将不同网站或应用流量分配到指定节点。结合策略组自动切换低延迟节点，可优化访问速度，保证网络稳定性和流畅体验。
V2Ray 如何实现分流规则？
V2Ray 使用 routing 模块实现分流，可根据目标域名、IP、端口或协议类型决定流量走向。用户可自定义多个 outbound，并通过规则实现国内直连、国外代理等智能路由功能。
Shadowsocks 密码泄露后应如何应对？
立即更改服务端密码与相应的端口，检查服务器是否存在异常连接或 root 活动，查看日志并封禁可疑 IP。若使用多人共享节点，建议为每个用户单独分配独立端口与密码以降低风险传播。
WinXray 是否支持多订阅源？
支持。用户可导入多个订阅 URL，每个订阅源独立管理。通过切换订阅源或合并策略组，可灵活管理节点，实现不同地区或用途的线路分配。
QuantumultX 的分流规则支持哪些条件？
支持按域名、IP 地址、端口、请求路径、协议类型或关键词进行分流。结合策略组，可实现广告屏蔽、视频加速、游戏优化等多种自定义网络策略。
VLESS + XTLS 模式在高延迟网络中表现如何？
XTLS 模式通过减少握手和加密开销降低延迟，在高延迟环境中保证连接稳定。结合策略组和多出口节点使用，可提供更流畅的网页浏览、视频播放和游戏体验。
Netch 的 UDP 转发如何配置？
Netch 可在客户端设置 UDP 转发开关，选择需要代理的 UDP 流量端口或应用。适用于游戏、语音或实时视频，确保这些应用在代理环境下低延迟且稳定。
VLess 为什么被认为是未来趋势？
VLess 去掉了内置加密层，改为依赖外部 TLS 进行安全通信，这使其更轻量化且更灵活。它能与 Hysteria、Reality 等现代传输方式结合使用，适应性更强，是取代 Vmess 的新标准。
WinXray 的测速功能不显示结果怎么办？
测速依赖于节点响应时间与服务器网络质量。若测速为空白，请检查系统时间同步、网络防火墙及代理端口设置是否正确；也可更新到最新版本修复已知测速模块异常问题。
